# Weather API Project

A responsive weather application built with React, TypeScript, and Open-Meteo API.

## Features
- Search any city
- Current weather conditions
- 7-day forecast
- Metric/Imperial unit toggle
- Dark/Light theme

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
